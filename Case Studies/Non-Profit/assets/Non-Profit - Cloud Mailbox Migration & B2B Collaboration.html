The Challenge

A health-focused Orange County non-profit ran mail on a reseller-hosted Office 365 tenant (GoDaddy-Hosted O365). The tenant model limited admin controls, complicated license management, and blocked the B2B cross-tenant collaboration the non-profit needed to work securely with a county public-health partner agency.

The non-profit also relied on remote clinical and case-management staff needing VPN access, endpoint compliance, and SharePoint-based file collaboration — all of which were held back by the reseller-tenant constraints and a patchwork of point tools.

  • Reseller-tenant mail (GoDaddy O365) limiting admin and governance
  • B2B cross-tenant blocker — can’t collaborate natively with partner agency
  • TLS mail flow with Mimecast + partner domain needed configuration
  • Remote / hybrid staff needing VPN, endpoint compliance, and SharePoint
  • Endpoint drift — Cisco AnyConnect, Umbrella, patches inconsistent
  • No single accountable IT partner across mail, endpoints, and collaboration
The Solution

Technijian delivered a two-track engagement: a formal mailbox migration off the reseller tenant to a direct Microsoft 365 tenant using MigrationWiz, followed by ongoing managed operations covering B2B partner setup, secure mail flow, endpoint compliance, and SharePoint collaboration support.

  • Mailbox migration off GoDaddy-Hosted O365 to native Microsoft 365 tenant using MigrationWiz — pre-stage + full delta pass, verified integrity
  • B2B cross-tenant setup with county public-health partner for secure collaboration on shared participant populations
  • Secure mail flow — TLS connection between Microsoft 365, Mimecast, and partner domain configured and validated
  • VPN & endpoint — Cisco Secure Client (AnyConnect), Cisco Umbrella, and ManageEngine patching kept current
  • SharePoint & file access — site membership management, access troubleshooting, and backup planning for recovery
  • Device compliance — Windows updates, feature packs, and configurations rolled through ManageEngine with verification
