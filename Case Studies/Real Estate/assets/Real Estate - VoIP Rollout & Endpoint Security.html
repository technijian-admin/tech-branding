The Challenge

A regional real estate operator ran a 30+ property portfolio with no consistent telephony standard. Each site had its own phone carrier, its own routing, and its own quirks. Staff turnover at property offices meant password policies, admin accounts, and endpoint security were inconsistent from site to site.

On top of that, a corporate IT assessment flagged significant security findings: stale admin privileges, inactive accounts, legacy operating systems still in use, FSMO roles consolidated on a single host, and Group Policy sprawl that no one fully understood.

  • No unified VoIP platform — 30+ sites on mixed carriers, inconsistent routing
  • Password policy gaps and stale admin accounts across the domain
  • Legacy OS instances still operational in production
  • FSMO single-point-of-failure — all roles on one domain controller
  • Group Policy sprawl with no documented ownership
  • Endpoint security drift — outdated CrowdStrike sensors on property PCs
The Solution

Technijian packaged the work into a multi-phase program: a portfolio-wide 3CX VoIP rollout executed property-by-property, followed by an infrastructure remediation track covering identity, endpoint security, and Active Directory structure. Every property received a standard VoIP configuration; every endpoint was brought current on CrowdStrike; every domain finding was tracked through remediation.

  • 33+ property VoIP cutovers on a standard 3CX configuration with documented call flows and DID planning
  • Identity remediation — password expiration policies, inactive account cleanup, admin privilege reduction
  • System & endpoint stability — reactivated AD/non-AD machines, decommissioned legacy OS installations
  • Active Directory restructure — redistributed FSMO roles, simplified and documented Group Policy
  • Endpoint security sweep — CrowdStrike sensor currency across property and corporate machines
  • Network & server optimization — addressed config drift, tuned storage, hardened web servers
