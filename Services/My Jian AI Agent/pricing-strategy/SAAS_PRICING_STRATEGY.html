The single most important finding Jian is priced against the wrong competitive set. V3.1 benchmarks MSP point-tools (Auvik $250, Liongard $50–100, Huntress $6). Jian's true category is agentic AIOps / AI-SOC, where comparable capability costs $20K–$378K/yr (Prophet $20–42K, Dropzone $36K, Torq $59–378K, Microsoft Security Copilot ~$105K, Arctic Wolf $80–96K, Moveworks/Aisera $90–130K+). A typical Jian client at $383–$5,822/mo sits at 5–25% of comparable — the value story is 2–5× stronger than the collateral claims.

The seven moves

  1. Keep the hybrid value metric (per-user + per-resource). Keep per-agent internal — it's a costing tool that sets resource prices, not a buyer metric.
  2. Re-anchor the value narrative to AI-SOC, not MSP tools. The receipts justify it.
  3. Monetize what's built but unsold — the mssql agent models ~$1,272/client/mo of labor value and has no SKU. Add Jian for SQL Server + Network/SNMP Monitoring now.
  4. Add a middle tier — $9 → $24 is a 2.7× cliff with no "land here". Insert Jian Plus $16/user/mo.
  5. Formalize the MSP / white-label wholesale program as a named SKU — the channel is wide open.
  6. Protect the small-account floor — bare 25-seat Core = $225 < $165–315 cost-to-serve. Set a $299/mo platform minimum.
  7. Hold list low for deliberate land-grab — then raise on a published cadence as reference customers accrue.

02The Value Metric — keep the hybrid, keep per-agent internal

The value metric is the single highest-leverage decision. Scored against the standard test, the two-axis hybrid wins: per-M365-user for the identity layer, per-monitored-resource for the infrastructure layer. The per-agent model fails the buyer-facing tests but is invaluable internally.

Why per-agent stays internal — but earns its keep: the floor→ceiling model is the right tool to set resource list prices and to catch under-monetized capability. It already surfaced the biggest gap — the mssql agent's ~$1,272/client/mo labor value with no retail SKU. Translate ceilings into resource prices; never put "per agent" on a quote.

Insights

  • Opacity is the norm — transparency is Jian's moat. 6/7 AI-SOC vendors hide prices; /pricing 404s are routine. Jian is the only player with published, verifiable, auto-true-up-able line items.
  • Nobody is openly priced below ~$10K/yr. Jian's $4.6K/yr entry owns undefended white space.
  • Opaque units (SCU/credit/Assist) prevent comparison. Jian meters on units the buyer counts — a trust win.

White space we take

  • Don't fight Microsoft on E5-bundled basic triage. Win on non-MS estates, cross-platform correlation, deeper remediation, consolidation.
  • Absorb compliance. Vanta/Drata charge $20–25K/yr to watch evidence Jian already holds — bundle it.
  • Incumbents price humans. Jian does the analyst work autonomously — far below their loaded rate at ~90% margin. Anchor to work eliminated.
