Technical Setup
Setup in 1–3 Business Days

Two API Paths.
Both Read-Only. Both Secure.

Essentials uses only the Sophos Central Partner API — no firewall configuration needed. Advanced/Enterprise additionally uses the XGS device-side REST API for full on-box log access.
Essentials Tier — Sophos Central Partner API Only
1
Sophos Central Partner credentials — OAuth2 client ID + secret from Partner Settings → API Credentials. Read-only scope. 5 minutes to generate.
2
Branding assets — logo (PNG, min 600px wide) and hex color codes for the Word template.
Advanced / Enterprise — Adds XGS Device-Side REST API
3
XGS REST API enabled — Administration → Device Access → Local Service ACL → enable HTTPS API. System → Administration → enable REST API. Supported on XGS firmware v18+.
4
Dedicated read-only API user — create a separate admin account on each XGS restricted to Technijian's public IP range (/32 block). We provide the IP list after onboarding.
5
Network access to port 4444 — Technijian infrastructure reaches https://<firewall>:4444/ via S2S VPN (recommended) or management port restricted to our IP range.
XGS REST API — Key Endpoints (read-only)
IPS eventsGET /api/v1/monitor/ips
ATP / malware callbacksGET /api/v1/monitor/atp
Web filter logsGET /api/v1/monitor/webfilter
Traffic analyticsGET /api/v1/monitor/interface
VPN tunnelsGET /api/v1/monitor/ipsec
Admin audit logGET /api/v1/monitor/admin
🔒 Read-Only. Always.
We never push configuration changes, acknowledge alerts on your behalf, or modify any firewall settings. All API calls are GET-only. Bearer tokens have a 30-minute TTL and are refreshed automatically by the pull script.
Network Access Options (Advanced+)
Recommended
S2S VPN: Permanent IPsec or SSL VPN tunnel from Technijian DC to client site — already configured for most managed clients.
Alternative
Management port restricted: HTTPS management on port 4444 restricted to Technijian /32 IP block only. Acceptable for clients without S2S VPN.
